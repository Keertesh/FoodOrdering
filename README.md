To run the application locally, follow these steps:

1. Frontend:
   - Navigate to the `frontend` folder.
   - Install dependencies: `npm install`
   - Run the application: `npm start`

2. Backend:
   - Navigate to the `backend` folder.
   - Install dependencies: `npm install`
   - Start the server: `npm start`
